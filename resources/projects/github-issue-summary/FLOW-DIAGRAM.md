```

Snapshot @ 3pm:     [#1, #5, #7, #9, #10]
                                    ^^^^
                               New issue appeared!
```

### 2. Delta = Difference Between Snapshots
```
Delta = Current - Previous
      = "What changed in the last 6 hours?"
```
